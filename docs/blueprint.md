# **App Name**: Supabase Data Lister

## Core Features:

- Supabase Connection: Connect to the Supabase project using credentials from .env files.
- Data Fetching: Fetch data from the 'indicators' table in Supabase.
- Data Listing: Display data from the Supabase table in a user-friendly table format, excluding JSON columns initially.
- .env Configuration: Setup .env files to store Supabase URL and API keys.
- Base64 Image Display: Display images from the 'image_mt5' column using a clickable icon, opening a modal or new tab.

## Style Guidelines:

- Primary color: Deep blue (#2C3E50) to reflect reliability and data focus.
- Background color: Light gray (#F0F4F8) for a clean and readable interface.
- Accent color: Teal (#3498DB) for interactive elements and highlights.
- Body font: 'Inter' sans-serif for readability.
- Headline font: 'Space Grotesk' sans-serif for a tech-focused and modern appeal.
- Simple, outline-style icons for actions like opening the base64 images, with a focus on clarity.
- Clean, table-based layout with clear separation of columns for easy data scanning.
